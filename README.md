
# Hierarchical Clustering 12 Kecamatan

## Deskripsi
Aplikasi ini dibuat menggunakan Streamlit untuk menampilkan hasil analisis Hierarchical Clustering pada 12 Kecamatan di Kota Bekasi berdasarkan dua variabel.

## Metode
- Hierarchical Clustering (Agglomerative Clustering)
- Linkage : Ward

## Variabel
1. Rata_Negeri
2. Rata_Swasta

## Jumlah Data
- 12 Kecamatan
- 2 Variabel
- 3 Cluster

## Fitur Aplikasi
- Menampilkan Dataset
- Diagram Batang
- Dendrogram
- Scatter Plot
- Hasil Clustering

## Library
- pandas
- matplotlib
- scipy
- scikit-learn
- streamlit
